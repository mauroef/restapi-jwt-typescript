# Instructions

## Create package.json

```
npm i -y
```

## Install typescript dependecy

```
npm i -D typescript
```

## Create typescript config file

```
npx tsc --init
```

## Install concurrently

```
npm i concurrently -D
```

## Install nodemon

```
npm i nodemon -D
```

## Install express (server)

```
npm i express -D
npm i @types/express -D
```

## Install morgan (view requests on console)

```
npm i morgan -D
npm i @types/morgan -D
```
